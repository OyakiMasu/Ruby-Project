# The Bakery App 

**Welcome to the Bakery App**

## Pre-Requisites
In order to use this repository you will need the following:

- Operating System **(Windows `10+`, Linux `3.8+`, or MacOS X `10.7+`)**
- RAM >= 4GB
- Free Space >= 2GB

## Built With
This application has been built with the following tools:

### CLIENT
- **React.js`v17.0.2`**   
- **Bootstrap `v5.3.0`**
- **Css `SSL`**

- **Ruby `v2.7.+`**
- **SQlite3 `v1.6`**
- **ActiveRecord `v7.0.4`**
